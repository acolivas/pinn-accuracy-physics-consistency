# Predictive accuracy diverges from physical consistency in physics-informed neural networks

This repository contains prototype code and sample data associated with the manuscript:

**Predictive accuracy diverges from physical consistency in physics-informed neural networks**

## Overview

This study examines how loss formulation, network architecture, and data availability affect predictive accuracy and physics consistency in physics-informed neural networks (PINNs) for indoor airflow reconstruction.

## Repository contents

- Prototype code for model training and evaluation
- Scripts related to loss and architecture ablation
- Sample dataset for demonstration

## Notes

This repository is intended to provide the core code structure and sample data associated with the study.

Additional data and full experiment outputs are available from the corresponding author upon reasonable request.
